# Protein Folding with VQE using Cirq

A quantum computing approach to protein folding prediction using Variational Quantum Eigensolver (VQE) implemented with Google's Cirq framework.

## Features

- **Multiple Interaction Models**: Hydrophobic, electrostatic, and van der Waals interactions
- **Variational Quantum Circuits**: Customizable ansatz with rotational and entanglement layers
- **Error Mitigation Ready**: Framework for implementing error mitigation techniques
- **Hybrid Optimization**: Combines classical optimization with quantum computation
